Set each alert using the same format as the timer above. Example: 25:30. This would trigger the TTS alert when the timer hits 25:30.

How this works

To use the timer by default, press start when the pre-war timer in game hits 1:00 minute exactly. You can edit the starting time to match whatever you want. If you want to run it when the pre-war timer hits 2:00 minutes, you'd set it to 32:00, etc.

The timer will run in the background and remind you of each alert you have configured above when the timer reaches the specified time.

When the war is complete the timer resets to your selected starting value. If you want to restore the original 31:00 setup and alert list, use Reset Timers to Defaults.
