How this works

To use the timer, press start when the pre-war timer in game hits 1:00 minute exactly.

The timer will run in the background and remind you of each spawn timer 30 seconds prior via a basic text-to-speech voice.

When the war is complete the timer should reset itself to 31 minutes (1 minute for pre-war timer, and 30 minutes for war). If a war ends earlier than 30 minutes you can just manually reset it with the button.
